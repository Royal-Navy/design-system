# Basic Example

```html
  <rn-input id="input" type="text" placeholder="Label"></rn-input>
```

#### Props

Prop Name       | Value      | Required
--------------- | ---------- | --------
**id**          | `{String}` | true
**placeholder** | `{String}` | true
**type**        | `{String}` | false

To use a textarea instead of an input, simply leave the `type` blank.
